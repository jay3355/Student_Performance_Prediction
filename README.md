
# Student Performance Prediction

## Project Description
A machine learning project that analyzes student academic data to predict performance outcomes and support early academic intervention.

## Dataset
- Student attendance
- Study hours
- Internal and test scores
- Engagement and behavioral metrics

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning
- Supervised learning
- Decision Tree Classifier
- Evaluation using accuracy and classification metrics

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python student_performance_prediction.py
